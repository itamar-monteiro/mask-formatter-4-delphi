# MaskFormatter4Delphi
Pequeno exemplo de um framework delphi para máscaras.
Exemplo de uso na pasta Demo.

#### Máscaras disponíveis:
- CPF/CNPJ
- CPF
- CNPJ
- Datas
- Telefone
- CEP
- Customizada (padrão ###-#####)

This is a small field validation framework for Delphi. An example of how to use it can be found in the "Demo" folder.
